# ChatGPT Clone

AI-powered chatbot built with Anthropic (Claude API Key).

## Features
- 5 personality modes (Helpful, Teacher, Motivator, Sarcastic, Coder)
- Adjustable creativity (temperature control)
- Adjustable reply length
- Chat history with token counter
- One-click clear chat

## Tech Stack
- Anthropic API
- Streamlit
- Python

## Run locally
pip install -r requirements.txt
streamlit run app.py
